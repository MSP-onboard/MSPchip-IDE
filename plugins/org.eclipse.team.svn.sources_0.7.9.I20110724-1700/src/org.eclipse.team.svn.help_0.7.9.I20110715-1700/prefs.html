Preferences

To configure preferences refer to 'Window>Preferences' menu item.

The default preferences customization is made according to the community advises. But of course each user can change it in the appropriate way to optimize his working cycle - actually this is the main goal of providing the user such rage of customized preferences. The global description of all of the preferences and recommendations on setting them is given inside this part of a guide. It's recommended to pay attention to this part because the optimized settings can seriously increase the users performance during his everyday work.
