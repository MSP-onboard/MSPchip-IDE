SVN Properties View

'SVN Properties View' provides a probability to view SVN® Properties for the resource. Its default location is 'SVN Repository Exploring Perspective'.

This is how 'SVN Properties View' looks like:

Toolbar

Refresh - redraws the view refreshing the information on SVN® Properties of the resource.

Link with Editor and Selection - shows in the view information on SVN® Properties of the selected or actively editing resource.

Pop-up menu

Save Value to File - saves the property value to a specified file.

Refresh - redraws the view refreshing the information on SVN® Properties of the resource.
