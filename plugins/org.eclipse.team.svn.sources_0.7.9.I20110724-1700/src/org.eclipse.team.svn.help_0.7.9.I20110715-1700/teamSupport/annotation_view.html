SVN Annotation View

'SVN Annotation View' provides a probability to view the history of file changes with exact highlighting of the changes from revision to revision. Its default location is 'SVN Repository Exploring Perspective'. By default 'History View' is also shown. To view the changes of specified revision in comparison to previous one simply select it's info in the table.
