Subversive management instructions

After Subversive installation the user can manage it using plug-in management options. Subversive can be disabled, enabled, uninstalled or updated. To manage Subversive:

  1. Start Eclipse and select menu item 'Help > About Eclipse', then press 'Installation Details' button.



  2. Next you'll see the dialog where you can select Subversive plug-ins in order to update, uninstall them or to simply check their properties.
